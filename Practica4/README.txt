Para acceder a los modos de dibujado con luces tenemos los siguientes botones:
	Tecla '1' - Puntos.
	Tecla '2' - Lineas.
	Tecla '3' - Solido.
	Tecla '4' - Solido Ajedrez.
	Tecla '5' - Visualizado con luz plana.
	Tecla '6' - Visualizado con luz suave.

Para acceder a las distintas figuras usamos las siguientes teclas:
	Tecla 'C' - Cubo
	Tecla 'O' - Objeto Ply
	Tecla 'P' - Pirámide
	Tecla 'F' - Trípode, modelo jerárquico.
	Tecla 'A' - Tanque

Luego tenemos distintas funciones dentro de nuestro modelo jerárquico "Trípode" (Tecla 'F'):
	Teclas 'F3'/'F4' - Girar la base del trípode a Izquierda/Derecha.
	Teclas 'F5'/'F6' - Ampliar el objetivo de la cámara para adelante y para atrás.
	Tecla 'F7' - muestra la animación de nuestro modelo jerárquico (Tecla 'F') junto con una luz que rota.
	Teclas 'F8'/'F9' - Subir/Bajar las patas del trípode.
	Tecla 'F10' - Cambiar los materiales de las figuras para las luces.
